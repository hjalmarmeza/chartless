# CHART LESS - Laberinto Silencioso

Una aplicación de meditación y agradecimiento diseñada para dispositivos móviles.

## Características
- **Laberintos Procedurales**: Nunca jugarás el mismo laberinto dos veces.
- **Árbol de la Vida Fractal**: Un árbol que crece visualmente con cada victoria.
- **Audio Atmosférico**: Sonido de olas sintético y música de fondo relajante ("lost in the ocean" por Limujii).
- **Video de Gratitud**: Fondo cinematográfico optimizado para Cloudinary con caché local para ahorro de datos.
- **Guía Luciérnaga**: Una ayuda visual si te quedas quieto por mucho tiempo.
- **Respiración Guiada**: El color del fondo pulsa para guiar tu respiración.

## Tecnologías
- HTML5 / Canvas API
- Web Audio API
- Cache API (para optimización de videos de Cloudinary)
- CSS3 (Aesthetics Primium)
